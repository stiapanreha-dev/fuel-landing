# Лендинг — продажа бензина АИ-92 / АИ-95

Одностраничный адаптивный сайт с доставкой топлива по Москве, МО и регионам России.

**Демо (GitHub Pages):** https://stiapanreha-dev.github.io/fuel-landing/

## Стек

- Статический HTML/CSS/JS
- Контент в YAML (`config/site.yaml`, `config/content.yaml`)
- Хостинг: GitHub Pages

## Локальный запуск

```bash
cd /path/to/fuel-landing
python3 -m http.server 8080
# Открыть http://localhost:8080
```

> Нужен локальный сервер: браузер загружает YAML через `fetch`.

## Как изменить бренд / контакты

1. Отредактировать `config/site.yaml` — компания, контакты, цвета, изображения.
2. Отредактировать `config/content.yaml` — тексты всех блоков.
3. Закоммитить и запушить — сайт обновится на GitHub Pages.

## Как заменить изображения

Положить файлы в `images/` и обновить пути в `config/site.yaml` и `config/content.yaml`.

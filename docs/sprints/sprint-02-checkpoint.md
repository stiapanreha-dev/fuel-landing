## Изменения в YAML

- `fuel.types[].badge`, `fuel.types[].fuel_select`
- `audience.items[]` — объекты `{ icon, label }`
- `audience.subtitle`

---

## Как добавить новый вид топлива

```yaml
fuel:
  types:
    - id: "ai98"
      name: "Бензин АИ-98"
      badge: "АИ-98"
      fuel_select: "АИ-98"
      description: "..."
      image: "images/fuel-ai98.jpg"
```

Опция автоматически появится в модальной форме.
